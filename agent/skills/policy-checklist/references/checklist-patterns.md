# 清单组织骨架

以下骨架只用于帮助组织答案结构，不能替代知识库检索。

## 1. 步骤清单

1. 确认事项是否适用当前示例政策。
2. 确认是否需要提前审批或事后补报。
3. 准备必需材料。
4. 提交给对应角色。
5. 跟进审批或补件结果。

## 2. 材料清单

- 基础信息： 【日期】、【地点】、【事由】、【金额】、【涉及人员】
- 凭证材料： 【发票/截图/行程单/诊断证明/照片/其他】
- 审批记录： 【直属经理/行政/财务/安全负责人审批】
- 补充说明： 【超标原因/紧急情况/例外情况】

## 3. 审批链

- 发起人： 【申请人】
- 第一审批角色： 【直属经理/团队负责人】
- 第二审批角色： 【HR/行政/财务/安全负责人】
- 备案对象： 【如适用】

## 4. 注意事项

- 是否存在提交时限
- 是否存在金额上限或次数限制
- 是否存在必须先审批后执行的要求
- 是否存在例外情形或补充材料要求
