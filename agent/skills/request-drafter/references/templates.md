# 示例模板骨架

以下模板骨架仅用于帮助组织语言。实际输出时，必须以本轮读取到的知识库内容为准。

## 1. 请假申请邮件

主题：请假申请 - 【姓名】 - 【日期/时段】

正文：

您好，【审批人称呼】。

我因【请假原因】，计划于【请假日期/时段】申请【假种，如适用】。根据当前示例公司政策，我将按要求完成【交接/系统提交/材料补充】。

如需补充材料或进一步审批，请告知。我会配合处理。

谢谢。

【姓名】
【部门/团队】
【日期】

## 2. 远程办公申请消息

您好，【审批人称呼】。

我申请于【日期】进行远程办公，原因是【原因】。当天我的工作安排包括：【工作内容】。

我会确保【在线时段/沟通方式/会议安排】符合当前示例政策要求。如需额外备案或审批，请告知。

## 3. 考勤异常说明

主题：考勤异常说明 - 【姓名】 - 【日期】

正文：

您好。

关于【日期】的考勤异常，情况说明如下：

- 异常类型： 【迟到/漏打卡/提早离开/其他】
- 具体时间： 【时间】
- 原因说明： 【原因】
- 已采取措施： 【补卡/联系主管/提交系统说明】

如按当前示例政策还需补充材料或走审批流程，请告知。

## 4. 报销说明

您好，【财务或审批人称呼】。

我提交本次【费用类型】报销说明，发生时间为【日期】，事由为【事由】，金额合计【金额】。

目前已准备材料包括：【发票/行程单/审批记录/其他】。如按当前示例政策还需补充附件、说明超标原因或追加审批，请告知。

## 5. 事件说明

您好。

现就【日期/地点】发生的【事件类型】说明如下：

- 发生时间： 【时间】
- 发生地点： 【地点】
- 经过概述： 【经过】
- 已采取措施： 【措施】
- 需要协助： 【需要协助的事项】

如按当前示例政策需要联系特定角色、补充记录或进一步上报，我会按要求处理。
