## 示例政策使用约定

- 回答前必须读取相关主题文件，不能只引用本索引。
- 回答必须明确说明内容是示例政策，而非用户所在公司的真实制度或法律意见。
- 答案应标注实际使用的文件及章节，并保留政策中的审批条件和例外。
- 涉及真实权益、当地法规、医疗判断、税务处理或紧急安全事件时，应提示用户向有权负责人或当地专业机构核实。
- 若问题不在上述范围，说明资料不足，并指出需要新增的 Markdown 主题。
