# PowerPoint 演示设计与交付策略

## 适用场景

适用于规划 PowerPoint 内容、控制幻灯片信息密度、准备演讲，以及在正式展示前排练并检查设备环境。

## 执行策略

### 设计服务于口头表达的幻灯片

1. 每页围绕一个主要信息组织内容，标题直接表达结论或该页作用。
2. 幻灯片文字应作为演讲提示而不是完整讲稿；如果观众一直盯着页面，内容可能过密或不够清晰。
3. 让数据、图表和颜色承担明确的信息作用，避免与核心信息无关的装饰。
4. 演示结构应有清晰开场、主体和结论，并预先规划各部分时间。

### 排练内容和节奏

1. 使用“排练计时”记录每页和整场时长；可在幻灯片浏览视图检查各页用时。
2. Microsoft 365 用户可使用 Speaker Coach 获取语速、语调、填充词、包容性语言以及是否照读幻灯片等反馈。
3. 根据排练报告修改讲法后再次排练。不要把 Coach 当作自动播放计时工具；它用于练习和反馈，不会写入幻灯片计时。
4. 紧张或语速过快时，在进入新页面或章节前停顿和深呼吸，并准备简短的过渡句。

### 正式展示前检查

1. 确认投影设备分辨率与制作演示文稿的计算机兼容，避免画面裁切。
2. 在实际屏幕上检查颜色、字号、图表可读性和媒体播放，并关闭屏幕保护程序。
3. 明确提问安排；如需要连贯讲完主体，可请观众把问题留到结尾。
4. 控制总时长，不要照读页面，也避免无意识移动鼠标指针分散注意力。
